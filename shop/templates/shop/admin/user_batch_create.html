批量创建用户

此功能仅限超级管理员使用，用于通过上传CSV文件批量创建用户。

CSV文件必须包含以下列：

  • username (必需): 用户的登录名，必须是唯一的。
  • password (必需): 用户的登录密码。
  • user_type (可选, 默认 'customer'): 用户类型。可以是 'customer', 'merchant', 或 'rider'。
  • shop_name (当 user_type 为 'merchant' 时必需): 如果创建商家账号，必须提供店铺名称。如果店铺不存在，系统将自动创建。
  • shop_description (可选): 如果创建新店铺，可以提供店铺的描述。
注意: 整个上传过程将在一个事务中完成。如果文件中任何一行数据有误，所有用户的创建都将被取消。
